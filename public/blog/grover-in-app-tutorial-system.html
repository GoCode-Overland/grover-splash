Grover Now Teaches You the App While You Use It

Published June 17, 2026 · 5 min read

Tags: app features · onboarding · user experience · Grover updates · vanlife app

Grover has a lot going on. There's the map with community pins, LandTrust properties, filters, and bucket list markers. There's the agent that plans routes, answers questions, and builds trip itineraries in seconds. There's pin creation, the Trophy Map, the Stats page, and everything in between.

That's a lot of amazing capability — and it can be hard to know where to start. We've been thinking hard about how to make sure every Grover user finds the features that are going to make their vanlife better, without making the app feel overwhelming or requiring a wall of tutorial text before you can go do the actual fun thing.

The answer we landed on: teach through doing.

Contextual Guidance, Right When You Need It

We've built a new in-app tutorial system that meets you exactly where you are. Rather than a one-time walkthrough you click through at setup and immediately forget, Grover now surfaces guidance at the moments it actually matters — when you're in the part of the app where a feature lives.

The system uses spotlight overlays: a focused highlight on a specific button or element, with everything else gently fading back. It's like having someone tap you on the shoulder and say "hey, try this" — except the thing they're pointing at is right there, and you can tap it immediately. No simulated interface, no separate tutorial mode. The spotlight points at the real feature. You tap it and it does exactly what it always does.

The tutorial and the feature are the same thing. When the spotlight highlights a button, you tap it and you're using the actual feature — no extra steps to find it yourself afterward.

The sequence moves forward as you go. Discover one thing, get a nudge toward the next. Already know something? Move on. The system doesn't get in your way — it clears the path in front of you.

Some of Our Favorite Moments in the Flow

The tutorial walks you through the features that unlock the most joy in Grover. A few highlights:

  • Your first bucket list marker. The tutorial shows you how to tap and hold on any spot on the map to save it to your bucket list — one of the most powerful (and satisfying) gestures in the app.
  • The + button for adding pins. The first time you add a pin to the community map is a big moment. The tutorial makes sure you find it.
  • Filter bar exploration. The map filters are where a lot of the magic lives — toggling layers, surfacing LandTrust properties, narrowing by place type. The tutorial pulls back the curtain.

Each of these moments is designed to be fast. You're not sitting through a lecture — you're tapping things and watching them work. That's the whole point.

A Tutorial That Keeps Getting Better

Grover is growing fast. New features ship all the time — the LandTrust layer, the Stats page, the bucket list completion flow — and we want to make sure every one of them gets properly introduced to the people who would love it most.

The tutorial system is built to grow with the app. As we ship new features, we can add them to the tutorial flow so new users discover them right away and existing users get a heads-up when something important changes. It's our way of making sure the best parts of Grover never stay hidden.

"The best tutorial gets out of the way the moment you understand what to do. We built ours around that idea."

What This Means for You

If you're new to Grover, the tutorial is your on-ramp. You'll see spotlight moments as you navigate the app for the first time — each one pointing you toward a feature worth knowing. By the time you've followed through the sequence, you'll have tried the things that make Grover genuinely useful, not just read about them.

If you've been using Grover for a while, keep an eye out for tutorial moments when new features land. We'll surface them contextually so you know something new is there and exactly how to use it.

Either way: open the app, start exploring, and let Grover show you what it can do. Our Getting Started guide is a great companion if you want a full overview before diving in.

We've put a lot of care into making Grover feel intuitive from the very first session. The tutorial system is one of the most important investments in that experience we've made — because the best vanlife app in the world is only useful if you actually know how to use it. Go find out what you've been missing.
